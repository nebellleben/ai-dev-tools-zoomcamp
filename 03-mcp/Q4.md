Use FastMCP to perform the web scraping task using python. Download content of any web page using Jina reader. To do so, simply add r.jina.ai in front of the address. For example to download http://datatalks.club, simply visit https://r.jina.ai/https://datatalks.club. Use the "requests" library.

Implement it in main.py.

To run it, use the following command:
uv --directory ~/03-mcp run python main.py
